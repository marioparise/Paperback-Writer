Paperback Writer

A simple eBook writing app built in Java.

Download this project as a .zip file Download this project as a tar.gz file

Paperback Writer is a Mac/Windows/Linux application that makes it easy for novelists to publish their ebooks as a valid ePub file.

As an aspiring author who has gone through the ebook publishing cycle, I was surprised by how difficult it is to create a good quality ebook. I ultimately had to dig into the code to create the quality I was seeking.

As I looked into why this was such a challenge, the answer became clear: Word processors like Microsoft Word or Apple's Pages (which are great) were never meant for ebook creation. They come with alot of baggage, such as handling hundreds of fonts, font sizes, tables, lists, columns, images and image placement... the feature list is huge. Any program that handles such a complex list of features must also have a very complex file format and structure to accomodate them. What's more, these programs are meant for the printed page. They deal with margins and page sizes and so forth - concepts that do not translate well to the electronic world.

Novels, by contrast, are simple. They deal with simple data: a table of contents, chapter titles, paragraphs. Sometimes you want to emphasize things with italics, bolds, or (much less likely for novels) underlines. That's about it.

Paperback Writer is a novel writing tool designed from the ground up for ebook publishing. It's a highly focused tool for a very specific purpose. It doesn't allow for any fancy formatting options. You only get what a novelist would typically require to write their story.

The result is a high-quality ebook that will work in your standard e-readers and can be submitted to self-publishing sites like Amazon or Lulu.
